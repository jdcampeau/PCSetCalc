# Prime Form Calculator in Python3

This calculator takes any list of musical notes and returns the prime form and the interval class vector for that set of pitches.
Click [here](https://musictheory.pugetsound.edu/mt21c/SetTheorySection.html) for more information on musical set theory.

## Getting Started

### Prerequisites

-Python 3
-Tkinter

### Installation

1. Clone the repository
```
git clone https://github.com/your-username/PCSetCalc.git
```
2. Grant run permission to 'main.sh':
```
chmod + x main.sh
```
3. Execute 'main.sh' to start:
```
./main.sh
```
